# ⚔️ Elden Ring Companion — Tarnished's Compendium

A beautiful item tracker for Elden Ring. Track every collectible: Talismans, Sorceries, Incantations, Ashes of War, Weapons, Armor, and more.

## Features
- 📦 150+ items across 11 categories
- ✅ Mark items as discovered (saved in localStorage)
- 📊 Progress tracking per category
- 🔍 Search and filter (Legendary, DLC, Found/Missing)
- 📷 Images from the Fextralife wiki
- 🎨 Dark Elden Ring-inspired UI
- 📱 Responsive design

## Deploy to Vercel
1. Push this repo to GitHub
2. Import on [vercel.com](https://vercel.com)
3. Deploy automatically

## Local Dev
```bash
npm install
npm run dev
```

## Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- localStorage for persistence
